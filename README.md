# Docker image for tello_ros2

## Build container
Just run inside repository folder:
```
./build.sh
```

## Run container
```
./start.sh
```

## Build packages
Enter /tello-ros2/workspace/ folder and run:
```
./../scripts/build.sh
```
## Run tello node
Follow instruction from the [original repository](https://github.com/tentone/tello-ros2)
